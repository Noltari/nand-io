Raspberry Pi Pico
=================

Tools needed
------------

```
cmake gcc-arm-none-eabi libnewlib-arm-none-eabi
```

RPi Pico Pinout
---------------

__Warning:__ always use 3.3V to power NANDs.

| Pico Pin        | NAND Pin | Debug Pin |
|:---------------:|:--------:|:---------:|
| GPIO0 (Pin 1)   |          | RX        |
| GPIO1 (Pin 2)   |          | TX        |
| GND (Pin 3)     |          | GND       |
| GPIO2 (Pin 4)   | I/O-0    |           |
| GPIO3 (Pin 5)   | I/O-1    |           |
| GPIO4 (Pin 6)   | I/O-2    |           |
| GPIO5 (Pin 7)   | I/O-3    |           |
| GND (Pin 8)     | GND      |           |
| GPIO6 (Pin 9)   | I/O-4    |           |
| GPIO7 (Pin 10)  | I/O-5    |           |
| GPIO8 (Pin 11)  | I/O-6    |           |
| GPIO9 (Pin 12)  | I/O-7    |           |
| GND (Pin 13)    | GND      |           |
| GPIO10 (Pin 14) | RE       |           |
| GPIO11 (Pin 15) | WE       |           |
| GPIO12 (Pin 16) | ALE      |           |
| GPIO13 (Pin 17) | CLE      |           |
| GND (Pin 18)    | GND      |           |
| GPIO14 (Pin 19) | WP       |           |
| GPIO15 (Pin 20) | RB       |           |
| GPIO16 (Pin 21) | CE       |           |
| GND (Pin 23)    | GND      |           |
| GND (Pin 28)    | GND      |           |
| 3.3V (Pin 35)   | VCC      |           |
| GND (Pin 38)    | GND      |           |
